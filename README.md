# BGCM — BoardGame Campaign Manager

Proyecto personal en Flask para visualizar y gestionar campañas de juegos de mesa.

## Objetivo

Tener una vista clara de campañas:

- Pendientes
- Activas
- Finalizadas o abandonadas

## Alcance actual (Fase 1)

- Gestión de juegos
- Gestión de expansiones
- Gestión de campañas y su estado

No incluye:

- Gestión de partidas
- Estadísticas
- Integraciones externas

## Tecnologías

- Python
- Flask
- Jinja templates

## Principios del proyecto

- Simplicidad por encima de completitud
- Formularios mínimos
- Claridad visual antes que estética
- Sin sobreingeniería
- Cambios incrementales y pequeños
